# Gerador de Conselhos 

Uma aplicação web interativa que consome a API pública Advice Slip para fornecer conselhos aleatórios, com tradução automática para o português utilizando processamento de linguagem natural.


## Funcionalidades
- Consumo de API realizando a busca de conselhos aleatórios em tempo real
- Traduz os conselhos do inglês para o português
- Interface Web com design moderno com efeitos de glassmorphism (fundo translúcido)
- Responsivo funciona bem em computadores e celulares

## Tecnologias Utilizadas
- Back: Python e Flask
- Front: HTML5 e CSS3 e um pouco de JavaScript

## APIs & Bibliotecas
- requests: Para chamadas HTTP
- deep-translator: Para integração com o Google Tradutor.
- Advice Slip API: Fonte dos conselhos originais.
